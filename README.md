# 🏠 Plan Your Perfect Stay

A web-based Property Booking and Management System developed using PHP and MySQL.

## 🚀 Features

* Admin Module
* Buyer Module
* Seller Module
* Property Management
* Property Booking
* Transaction Management
* Image Upload
* Google Maps Integration
* User Authentication

## 🛠️ Technologies Used

* PHP
* MySQL
* HTML5
* CSS3
* Bootstrap 5
* JavaScript
* Google Maps API

## 📂 Project Structure

admin/
buyer/
seller/
property/
transaction/
uploads/
include/

## ⚙️ Installation

1. Clone Repository
2. Copy project to XAMPP htdocs folder
3. Import pps.sql in phpMyAdmin
4. Start Apache and MySQL
5. Open:
   http://localhost/plan_your_perfect_stay
